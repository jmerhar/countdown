Countdown Solver

A web application for solving the words and numbers game on the long-running ITV game show Countdown.

Instructions

This application can solve both the Numbers and Letters games from Countdown. To begin, click on either the 'Numbers game' or 'Letters game' tab below. Follow the instructions for the game you've chosen to select your numbers or letters, then press the 'GO' button to start the solver.

Numbers Game
  1. Select any 6 numbers from the board.
  2. Enter a 3-digit target number in the "Target" box.
  3. Press the GO button to start finding solutions.
Letters Game
  1. Select any 9 letters from the board.
  2. Press the GO button to start finding words.

You can press the STOP button at any time to cancel the current calculation.
